PROJECT DESCRIPTION:
Currency Converter Application that converts currencies using real-time exchange rates.

API USED:
Name: Frankfurter API
Documentation: https://api.frankfurter.app/
Description: Free currency exchange rates API with no authentication required

INSTRUCTIONS TO RUN:
1. Open the project folder in a web browser
2. Navigate to index.html for the portfolio homepage
3. Click "Go to Currency Converter" or navigate directly to app.html
4. Enter an amount and select currencies to convert
5. Click "Convert Currency" to see the result

TECHNICAL DETAILS:
- HTML: Portfolio homepage (index.html) and main app (app.html)
- CSS: Responsive styling with modern layout
- JavaScript: API integration using fetch() and async/await
- API Endpoints Used:
  - https://api.frankfurter.app/currencies (get currency list)
  - https://api.frankfurter.app/latest (get exchange rates)

FEATURES:
- Real-time currency conversion
- Support for multiple currencies
- Input validation
- Error handling for API failures
- Clean, responsive UI
- Swap currencies functionality
